# React User Management App

## 📌 Project Overview
This is a React-based User Management Application built using Vite. It allows users to add, validate, display, and manage user data with form validation and API integration using Axios.

---

## 🚀 Features
- Add new users using a form
- Form validation (required fields check)
- Display user list dynamically
- API integration using Axios
- Responsive and clean UI
- Component-based React architecture

---

## 🛠️ Tech Stack
- React.js
- Vite
- JavaScript (ES6+)
- Axios
- HTML & CSS
